# Documentación para el asistente (RAG)

Este archivo es un ejemplo. Puedes añadir aquí guías breves sobre tu app:

- Cómo publicar una solicitud y qué campos son obligatorios
- Cómo enviar/aceptar ofertas y qué estados existen
- Qué endpoints están disponibles y qué devuelven

El índice se genera con `python manage.py build_assistant_index` y el chat
usará los fragmentos más relevantes como contexto.
